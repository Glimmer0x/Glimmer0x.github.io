# 个人网站设计方案构思

## 方案一：Editorial Brutalism（编辑式粗野主义）

<response>
<text>
**设计运动**：Editorial Brutalism — 受印刷排版与瑞士国际主义风格影响，融合当代数字粗野主义美学。

**核心原则**：
1. 强烈的排版层次：标题使用超大字号（clamp 80px~160px），正文使用精细的衬线体，形成戏剧性对比
2. 不对称网格：打破传统居中布局，采用偏移式构图，文字与图像不规则交错
3. 原始质感：轻微纸张纹理背景，粗边框线条，无圆角或极小圆角
4. 单色主导：以近黑色（#0D0D0D）与米白（#F5F0E8）为主，搭配一个强烈的强调色（如荧光黄 #E8FF00）

**色彩哲学**：
- 背景：#F5F0E8（温暖米白，模拟印刷纸张）
- 前景：#0D0D0D（近黑，非纯黑）
- 强调：#E8FF00（荧光黄，用于悬停状态、标签、CTA按钮）
- 情感意图：严肃、知性、有力量感，同时带有一丝叛逆

**布局范式**：
- 导航栏：左侧固定名字，右侧稀疏导航项，无背景色
- Hero：全屏高度，名字以超大排版占据左侧70%，右侧留白或放置抽象线条图形
- 内容区：三列不等宽网格，打破对称性
- 项目展示：横向滚动卡片，卡片边框粗重

**标志性元素**：
1. 粗重的水平分割线（2-3px solid）作为章节分隔
2. 大写字母标签（ABOUT / WORK / CONTACT）配合字母间距拉伸
3. 鼠标悬停时文字下划线动画（从左到右展开）

**交互哲学**：克制而有力，悬停效果使用颜色反转（黑底白字 ↔ 白底黑字），无过多动效

**动效**：
- 页面进入：文字逐行从下方滑入（stagger 0.1s）
- 滚动视差：Hero标题轻微向上漂移
- 悬停：背景色块快速填充（0.15s ease）

**字体系统**：
- 标题：Playfair Display（衬线，戏剧感）
- 正文：DM Sans（无衬线，现代可读）
- 标签：Space Mono（等宽，技术感）
</text>

## 方案二：Quiet Luxury Minimalism（静奢极简主义）

<response>
<text>
**设计运动**：Quiet Luxury — 受高端时装品牌（如 The Row、Bottega Veneta）视觉语言启发，强调克制与精致。

**核心原则**：
1. 极度留白：内容密度低，每个元素都有充足的呼吸空间
2. 精细的排版层次：字重差异微妙（300 vs 400 vs 500），而非大幅度跳跃
3. 材质感：细腻的米色背景、半透明玻璃卡片、极淡的阴影
4. 内敛的动效：一切过渡都缓慢而优雅（0.4s~0.6s ease）

**色彩哲学**：
- 背景：#FAFAF8（接近白色的暖灰）
- 前景：#1A1A1A
- 强调：#8B7355（暖棕金，低饱和度）
- 次要：#C4B9A8（浅沙色，用于边框和分割线）
- 情感意图：高端、沉静、值得信赖，传递专业品味

**布局范式**：
- 导航：极简顶部导航，仅文字，无背景，滚动后出现细线下边框
- Hero：左右分栏，左侧文字（名字+职位+简短介绍），右侧放置精心生成的肖像风格图像
- 内容区：宽松的单列或两列布局，大量留白
- 时间线式经历展示：竖向时间线，左侧年份，右侧内容

**标志性元素**：
1. 极细的线条（0.5px~1px）作为装饰和分隔
2. 小写字母为主的排版风格（反传统的全大写）
3. 数字以大字号展示（如技能百分比、项目数量）

**交互哲学**：如丝般顺滑，一切都是渐变过渡，无突兀跳变

**动效**：
- 滚动触发：元素淡入（opacity 0→1，translateY 20px→0，duration 0.6s）
- 导航悬停：下划线从中心向两侧展开
- 卡片悬停：极轻微上移（translateY -4px）+ 阴影加深

**字体系统**：
- 标题：Cormorant Garamond（优雅衬线）
- 正文：Jost（几何无衬线，干净现代）
- 数字：Cormorant Garamond（衬线数字更具美感）
</text>

## 方案三：Neo-Brutalist Terminal（新粗野主义终端风）

<response>
<text>
**设计运动**：Neo-Brutalism + Terminal Aesthetic — 受黑客文化、命令行界面与当代新粗野主义网页设计影响。

**核心原则**：
1. 暗色为主：深色背景（#0A0A0F），模拟终端屏幕
2. 荧光强调：使用绿色（#00FF88）或青色（#00D4FF）作为强调色，模拟磷光屏幕
3. 等宽字体主导：大量使用 Monospace 字体，增强技术感
4. 粗边框与硬阴影：卡片使用 2-3px 实线边框，阴影为硬阴影（无模糊）

**色彩哲学**：
- 背景：#0A0A0F（近黑深蓝）
- 前景：#E8E8E0（暖白）
- 强调一：#00FF88（荧光绿）
- 强调二：#FF3366（品红，用于错误/警告状态）
- 情感意图：技术极客、前沿、充满能量，适合开发者或技术从业者

**布局范式**：
- 导航：顶部固定，左侧显示 `~/liangxi` 路径样式，右侧导航项带 `>` 前缀
- Hero：全屏终端风格，打字机动效逐字显示介绍文字，光标闪烁
- 内容区：卡片式网格，卡片带硬边框和偏移硬阴影
- 技能展示：进度条样式，使用 ASCII 风格 `[████████░░]`

**标志性元素**：
1. 打字机光标动效（`|` 闪烁）
2. 代码注释风格的说明文字（`// about me`）
3. 终端提示符前缀（`$ ` 或 `>` ）

**交互哲学**：快速响应，带有轻微"扫描线"或"故障"效果，悬停时出现荧光色边框

**动效**：
- Hero：打字机逐字显示效果（每字符 50-80ms）
- 滚动触发：元素从左侧滑入，模拟终端输出
- 悬停：荧光色边框闪烁一次后保持

**字体系统**：
- 标题：JetBrains Mono（等宽，专为开发者设计）
- 正文：IBM Plex Mono（等宽，可读性强）
- 强调文字：Rajdhani（科技感无衬线）
</text>

## 选定方案

**选定：方案二 — Quiet Luxury Minimalism（静奢极简主义）**

理由：对于综合型个人网站，静奢极简风格能够跨越职业背景，展现专业素养与个人品味，不局限于特定行业。优雅的排版、充足的留白和精致的动效能够让访客将注意力集中在内容本身，同时传递出高端、可信赖的个人品牌形象。
